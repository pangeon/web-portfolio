Programowanie webowe

Mieszkam w Pabianicach. Lubię czytać książki, spędzać czas na grach strategicznych, czasem pisać wiersze. Rozwijam się od kilku lat jako projektant stron internetowych, zgłębiam zagadnienia z zakresu nowoczesnych technologii w obszarze: projektowania graficznego i programowania.

Swoją pierwszą aplikację napisałem w ActionScript 3 z drobną pomocą interfejsu graficznego aplikacji Adobe Flash. Od zawsze fascynowały mnie technologie internetowe, więc zacząłem rozwijać się jako webdeveloper. W ciągu kilku lat zglębiłem wiele metodologii i języków programowania. WEB-DESIGN TOOLS

Interaktywna strona WWW w technologii HTML5

Sposoby budowania interaktywnych stron WWW w technologii HTML5, z wykorzystaniem narzędzi firmy Adobe.

Witrynę wykonałem jako projekt zaliczeniowy studiów podyplomowych: Grafika i Techniki Multimedialne w Instytucie Informatyki Stosowanej Politechniki Łódzkiej Posłużyłem się najprostszą, a zarazem najbardziej fundamentalną metodą: stworzyłem witrynę odręcznie. Zapoznałem się z każdą linijką kodu, który osadzałem.

Projekt, który prezentuje wykorzystuje najbardziej podstawowe technologie webowe - HTML5 (połączenie funkcjonalności XHTML i możliwości JavaScriptu) oraz CSS3. Odwołuje się w nim do zainteresowań związanych z projektowaniem graficznych przy pomocy ulubionych narzędzi firmy Adobe. Zamieściłem w nim obszery katalog prac różnych twórców oraz kilka swoich.

SEKP

System eweidencji kompetencji pracownika

Aplikacja webowa napisana przy użyciu platformy Java Enterprise. Projekt na zaliczenie studiów podyplomowych "Nowoczesne aplikacje biznesowe w Java EE". Stos technologiczny: EJB, Java Persitence Api, MySQL, JSP, JavaScript. Narzędzia: NetBeans, GlassFish, Maven

Aplikacja o architekturze MVC zapewniającej jej łatwość w konserwacji i skalowalność. System umożliwia agregację i edycję danych użytkowników wraz z poziomami uprawnień. Zadaniem programu jest przechowywanie danych kompetencjach pracowników firmy, takich jak: historia zatrudnienia, spis ukończonych uczelni oraz szkoleń, rodzaje nabytych umiejętności. Witryna pozwala na filtrowanie wyświetlonych danych celem szybkiego wyszukania istotnych danych.

W części praktycznej pracy zrealizowany został system informatyczny umożliwiający gromadzenie informacji o pracownikach i kwalifikacjach przez nich nabywanych, a takż e wyszukiwanie pracowników m. in. w oparciu o w/w dane. System ten został zrealizowany w formie aplikacji na platformę Java EE zgodnie z tematyką studiów. Aplikacja realizuje założone funkcje w sposób platformę Java EE zgodnie z tematyką studiów. Aplikacja realizuje założone funkcje w sposób poprawny. Warto nadmienić użycie przez Autora nowoczesnego Criteria API do konstruowania złożonych zapytań bazodanowych.

JMTrain

Zarządzanie bazą szkoleń.

Program napisany w Javie z wykorzystaniem bibliotek frameworka Spring oraz baz danych MySQL. Prosty layout panel zarządzania napisany z wykorzystaniem Bootstrapa.

Aplikacja pozwala na zarządzanie szkoleniami. Program umożliwia edycję danych osobowych użytkowników oraz dodawanie zapisów na kursy. System jest bezpieczny - wysyła potwierdzenie e-mailem po rejestracji oraz chroni hasła. Szyfruje wrażliwe dane przy użyciu algorytmu MD5.
